═══════════════════════════════════════════════════════════════
ONE-LINER COMMANDS TO DOWNLOAD & EXECUTE FROM GITHUB
═══════════════════════════════════════════════════════════════

POWERSHELL ONE-LINER (Copy & Paste - PowerShell 5.x Compatible):
───────────────────────────────────────────────────────────────
powershell.exe -ExecutionPolicy Bypass -Command "[Net.ServicePointManager]::SecurityProtocol=[Net.SecurityProtocolType]::Tls12;$content=(Invoke-WebRequest -Uri 'https://raw.githubusercontent.com/monobrau/windows-network-config/main/duoproxyupdate/DuoProxyUpgrade.ps1' -UseBasicParsing).Content;$file='$env:TEMP\DuoProxyUpgrade.ps1';Out-File -FilePath $file -InputObject $content -Encoding UTF8 -NoNewline;& $file"

POWERSHELL ONE-LINER (Shorter - Direct Execution - PowerShell 5.x Compatible):
───────────────────────────────────────────────────────────────
powershell.exe -ExecutionPolicy Bypass -Command "[Net.ServicePointManager]::SecurityProtocol=[Net.SecurityProtocolType]::Tls12;iex((Invoke-WebRequest -Uri 'https://raw.githubusercontent.com/monobrau/windows-network-config/main/duoproxyupdate/DuoProxyUpgrade.ps1' -UseBasicParsing).Content)"

ALTERNATIVE (Download to file first - bypasses cache issues):
───────────────────────────────────────────────────────────────
powershell.exe -ExecutionPolicy Bypass -Command "[Net.ServicePointManager]::SecurityProtocol=[Net.SecurityProtocolType]::Tls12;$ts=Get-Date -Format 'yyyyMMddHHmmss';Invoke-WebRequest -Uri 'https://raw.githubusercontent.com/monobrau/windows-network-config/main/duoproxyupdate/DuoProxyUpgrade.ps1' -OutFile \"$env:TEMP\DuoProxyUpgrade_$ts.ps1\" -UseBasicParsing;powershell.exe -ExecutionPolicy Bypass -File \"$env:TEMP\DuoProxyUpgrade_$ts.ps1\""

CMD/BATCH ONE-LINER (PowerShell 5.x Compatible):
───────────────────────────────────────────────────────────────
powershell.exe -ExecutionPolicy Bypass -Command "[Net.ServicePointManager]::SecurityProtocol = [Net.SecurityProtocolType]::Tls12; Invoke-WebRequest -Uri 'https://raw.githubusercontent.com/monobrau/windows-network-config/main/duoproxyupdate/DuoProxyUpgrade.ps1' -OutFile %TEMP%\DuoProxyUpgrade.ps1 -UseBasicParsing; powershell.exe -ExecutionPolicy Bypass -File %TEMP%\DuoProxyUpgrade.ps1"

═══════════════════════════════════════════════════════════════
USAGE:
───────────────────────────────────────────────────────────────
1. Copy one of the commands above
2. Paste into PowerShell or CMD prompt
3. Press Enter
4. Script downloads and executes automatically

═══════════════════════════════════════════════════════════════
NOTES:
───────────────────────────────────────────────────────────────
- All commands are compatible with PowerShell 5.1+ (Windows 10/11 default)
- TLS 1.2 is enabled automatically for GitHub HTTPS connections
- Second command includes cache-busting (random number) to ensure fresh download
- Update GitHub URL if repository/branch differs
- First command downloads to temp folder then executes
- Second command executes directly from memory (no temp file)
- Third command works from CMD/Batch files
- If you get cached/old version, see CLEAR_CACHE.md for troubleshooting
